# L'archive comme outil de production

Site Vite + React pour un projet de mémoire en design graphique.

## Installation

```bash
npm install
npm run dev
```

Les vraies photographies peuvent être placées dans `public/photos/` puis référencées dans `src/data/images.js`.
Le fichier attendu pour l'écran d'entrée est `public/photos/drawer-front.jpg`.
